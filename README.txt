INTRODUCTION
------------

 * Drupal 8: The Cart Flyout is a drop-in replacement of the cart block provided
   by Drupal Commerce. This provides a sidebar which will "flyout".


INSTALLATION
------------

It is recommended to install this module using Composer.

 * Install the Commerce Cart Flyout module as you would normally install a
   contributed Drupal module. Visit https://www.drupal.org/node/1897420 for
   further information.


REQUIREMENTS
------------

These modules are required for commerce cart flyout.

 * Commerce - https://www.drupal.org/project/commerce
 * Commerce Cart Api - https://www.drupal.org/project/commerce_cart_api


CONFIGURATION
-------------

 * For complete documentation including configurations, please visit
   https://www.drupal.org/docs/8/modules/commerce-cart-flyout
